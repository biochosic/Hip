Франсуа Вієт

Народився 1540 року. Геніальний математик і дешифрувальник, який ввів літерні позначення в алгебру.

x₁ + x₂ = -p, x₁ · x₂ = q

Рене Декарт

Засновник аналітичної геометрії. Ввів термін «дискримінант» для розрізнення кількості коренів.

D = b² - 4ac

Практичний Тренажер

x² - 6x + 9 = 0. Знайти D.
D = (-6)² - 4*1*9 = 36 - 36 = 0.
x² + 3x + 5 = 0. Скільки коренів?
D = 3² - 4*1*5 = 9 - 20 = -11. D < 0, коренів немає.
2x² - 5x + 2 = 0. Знайти D.
D = (-5)² - 4*2*2 = 25 - 16 = 9.
x² - 4x + 4 = 0. Скільки коренів?
D = (-4)² - 4*1*4 = 0. Один корінь.
x² - 8x + 12 = 0. Сума та добуток?
Сума: 8, Добуток: 12.
Корені -2 та 5. Скласти рівняння.
x² - ( -2+5 )x + (-2*5) = x² - 3x - 10 = 0.
x² + 10x + 21 = 0. Знайти корені.
x1 = -3, x2 = -7 (бо -3-7 = -10, -3*-7 = 21).
x² - 5x - 14 = 0. Сума та добуток?
Сума: 5, Добуток: -14.
x² - 4x + 3. Знайти корені та розкласти.
Корені 1 та 3. Розклад: (x - 1)(x - 3).
x² + 5x + 6. Розкласти на множники.
Корені -2 та -3. Розклад: (x + 2)(x + 3).
x² - 2x - 8. Подати у вигляді добутку.
Корені 4 та -2. Розклад: (x - 4)(x + 2).
x² + x + 1. Чи можна розкласти?
D = 1 - 4 = -3. Розкласти не можна, бо немає дійсних коренів.
